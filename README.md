# 🎵 YouTube 轉 MP3 隨身版 (YouTube to MP3 Portable Downloader)

這是一個基於 Python 與 Tkinter 開發的輕量級、免安裝的 YouTube 轉 MP3 工具。具備圖形化介面（GUI），並採用**「雙引擎獨立外掛」**架構，完美解決了傳統打包軟體容易因為 YouTube 演算法更新而永久失效的痛點。

## ✨ 核心特色功能

* **🚀 一鍵極速下載**：貼上網址即可一鍵下載 YouTube 影片並自動轉換為 192kbps 的高音質 MP3。
* **🛠️ 雙引擎自動配置**：主程式極度輕巧。首次使用只需點擊按鈕，程式會自動連線至 GitHub 抓取最新的 `FFmpeg` 與 `yt-dlp` 核心引擎。
* **🔄 突破封鎖（熱更新機制）**：若未來 YouTube 更改演算法導致下載失敗，不需重新安裝軟體，只需點擊介面上的「更新 yt-dlp」，程式即可自動升級破解核心。
* **📊 即時進度監控**：內建進度條與狀態列，精準顯示下載百分比與轉檔狀態，告別盲目等待。
* **💾 智慧記憶設定**：自動記憶上次的下載儲存位置，並具備「一鍵開啟資料夾」與「視窗置頂」的貼心功能。

## 📥 如何使用 (一般使用者)

1. 前往本專案的 [Releases 頁面] (您未來若發布 exe 可將網址貼在此處) 下載最新的 `.exe` 執行檔。
2. 雙擊打開軟體，若為首次使用，請依序點擊介面上的 **「下載 FFmpeg」** 與 **「更新 yt-dlp」**。
3. 引擎配置完成後，貼上 YouTube 網址，選擇儲存位置，點擊 **「一鍵下載 MP3」** 即可！

*(註：程式會自動在同目錄下生成 `config.json` 記憶您的設定，請將 `.exe` 與設定檔放在一起使用)*

## 💻 開發者指南 (從源碼執行)

如果您想自行修改程式碼或打包，請確保您的電腦已安裝 Python 3.x。

**1. 安裝必要套件：**
此專案主要使用 Python 內建標準庫，介面使用 `tkinter`。唯一需要安裝的第三方打包套件為 `pyinstaller`。

```bash
pip install pyinstaller
```
2. 執行程式：
```bash
python ytmp3_gui.py
```
3. 打包為單一 EXE 檔：
```bash
pyinstaller --noconsole --onefile ytmp3_gui.py
```
(打包後的檔案將生成於 dist 資料夾中)
